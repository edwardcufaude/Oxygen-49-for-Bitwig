GENERIC MIDI KEYBOARD WITH 8 KNOBS

This script is a simple Midi input script to be used with any midi keyboard that features 8 knobs or faders that send CCs.

Just add it manually to the controller list and set up the input/output of your Midi interface or your USB Midi Keyboard in order to use it to play instrument devices and control Macros in Bitwig Studio.

In order to work correctly, you must set up your Midi keyboard to send the Midi CC# 20-27 when using the knobs/faders.
